WHAT IS Onsen UI?

Onsen UI is a web components UI framework optimized for PhoneGap/Cordova hybrid app development.

With the next version of Onsen UI, development utilizing our Onsen CSS Components will be possible, and on top of that you will also be able to integrate important JS frameworks like jQuery and Backbone.js.

With Onsen UI, developers can freely and easily make beautiful hybrid app UIs - like native apps.
