WHAT IS Onsen UI?

Onsen UI is a web components UI framework optimized for PhoneGap/Cordova hybrid app development.

Onsen UI makes possible a full stack of mobile app development tools by bringing together first our Onsen CSS Components along with a UI builder and debugger, build functionality, and more.
