ons-loading-placeholder

Onsen UIが読み込まれるまでに表示するプレースホルダーを表現します。

例

<div ons-loading-placeholder="page.html">
  Loading...
</div>

属性

ons-loading-placeholder 読み込むページのURLを指定します。
