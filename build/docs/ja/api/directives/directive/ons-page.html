ページ定義のためのコンポーネントです。このコンポーネントの内容はスクロールが許可されます。

属性

var このページを参照するための変数名を指定します。
modifier スタイル定義をカスタマイズするための名前を指定します。
onDeviceBackbutton デバイスのバックボタンが押された時の挙動を設定できます。
ngDeviceBackbutton デバイスのバックボタンが押された時の挙動を設定できます。AngularJSのexpressionを指定できます。

メソッド

getDeviceBackButtonHandler() ons-pageに紐付いているバックボタンハンドラを取得します。このメソッドはnullを返す場合があります。
