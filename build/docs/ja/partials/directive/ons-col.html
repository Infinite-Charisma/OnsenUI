ons-col

View This Code

グリッドシステムにて列を定義します。ons-rowとともに使用し、コンポーネントの配置に使用します。

  • Usage
  • Live Example
  • See Also

Usage

<ons-row>
  <ons-col width="50px"><ons-icon icon="fa-twitter"></ons-icon></ons-col>
  <ons-col>Text</ons-col>
</ons-row>

注意: Android 4.3以前、もしくはiOS 6以前のOSの場合、ons-rowとons-columnを組み合わせた場合に描画が崩れる場合があります。
