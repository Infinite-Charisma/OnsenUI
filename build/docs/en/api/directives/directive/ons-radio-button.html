ons-radio-button

Radio button component

Attributes
ng-model The model to bind to
value The value to be set to the model when this button is selected
left-label The label to be shown on the left of the button
right-label The label to be shown on the right of the button
name The group name of this radio button

This tag has no method

Known issue: Square box may be displayed for radio button in Android 2.3
