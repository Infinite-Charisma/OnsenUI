Should be used as root component of each page. The content inside page component is scrollable.

Attributes

var Variable name to refer this page.
modifier Specify modifier name to specify custom styles.
onDeviceBackbutton The onDeviceBackbutton attribute allows you to specify custom behavior when device's back-button is tapped.
ngDeviceBackbutton The ngDeviceBackbutton attribute allows you to specify custom behavior with AngularJS expresion when device's back-button is tapped.

Methods

getDeviceBackButtonHandler() Retrieve the back-button handler. This method may returns null.
